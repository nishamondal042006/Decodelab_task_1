# Personal CV Website

## Project Overview

* Developed a personal CV website using HTML5 and CSS3.
* Created a professional platform to showcase educational background, technical skills, projects, certifications, and contact information.
* Designed a clean and structured interface to improve readability and user experience.
* Implemented a responsive layout to ensure compatibility across multiple devices.

## Technologies Used

* HTML5
* CSS3

## Key Features

* Responsive and mobile-friendly design.
* Professional resume-style webpage layout.
* Organized sections for profile, education, skills, projects, and certifications.
* Simple and intuitive navigation.
* Clean visual design with consistent styling.
* Easy-to-maintain and scalable code structure.

## Skills Demonstrated

* Front-End Web Development
* Responsive Web Design
* HTML Structuring
* CSS Styling and Layout Design
* UI/UX Design Principles
* Code Organization and Maintainability

## Learning Outcomes

* Strengthened understanding of semantic HTML and modern CSS practices.
* Gained hands-on experience in designing responsive web interfaces.
* Improved webpage structuring and styling techniques.
* Enhanced ability to create professional and visually appealing web applications.
* Developed practical experience in building portfolio and resume-based websites.
